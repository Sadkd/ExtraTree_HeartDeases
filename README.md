# ❤️ Heart Disease Prediction using Extra Trees Classifier

This project uses Machine Learning to predict heart disease based on patient medical data.
The model is built using Extra Trees Classifier and includes data preprocessing, visualization, training, and evaluation.

## 📌 Project Overview

This notebook performs the following steps:
  * Load and explore heart disease dataset
  * Data cleaning (handling missing values)
  * Encoding categorical variables
  * Feature scaling
  * Train/Test split
  * Model training using Extra Trees Classifier
  * Model evaluation and accuracy measurement
  * Data visualization

## 📂 Dataset
The dataset used: heart_disease_uci.csv
Contains patient health attributes such as:
  * Age
  * Sex
  * Chest Pain Type
  * Cholesterol
  * Target variable: num (Heart Disease presence)

## 🧠 Model Used
Extra Trees Classifier from sklearn.ensemble:
 * Reduces overfitting
 * Good performance for classification problems
